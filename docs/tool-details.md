# 各ツールの詳細

アバターのアップロードや改変に使用するツールの最低限の解説です。

他のツールを知りたい人は[こちらのページ](/my-tools)を参照してください。


## 🎨 描画・見た目に関するツール

* **lilToon** [公式サイトを開く](https://lilxyzw.github.io/lilToon/ja_JP/)  
  アバターの見た目を設定する必須のシェーダー。<br>
  これがないとアバターが正しく表示されない。

## ⚙️ 改変・ギミックに関するツール

* **Modular Avatar** [公式サイトを開く](https://modular-avatar.nadena.dev/ja/)  
  衣装の着せ替えを非破壊で行えるツール。現在のVRC改変に不可欠。

* **AvatarMenuCreator for Modular Avatar** [BOOTHページを開く](https://booth.pm/ja/items/4419509)  
  衣装のON/OFFスイッチなどを簡単に作成できる補助ツール。

* **FaceEmo** [BOOTHページを開く](https://booth.pm/ja/items/4915091)  
  アバターの表情を直感的に設定・作成できるツール。

## 📦 その他

* **VRCQuestTools** [BOOTHページを開く](https://booth.pm/ja/items/2436054)  
  PC用アバターをQuest（Android）向けに変換・最適化するツール。

* **Avatar Optimizer** [BOOTHページを開く](https://booth.pm/ja/items/4885109)  
  アバターのデータサイズを軽量化し、パフォーマンスを向上させるツール。
